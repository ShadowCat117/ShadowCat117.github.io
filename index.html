James Callan Projects

These are projects I either actively work on, or worked on throughout the entire development lifecycle.

Wynntils

Wynntils is a rewrite of the original Minecraft mod Wynntils but for modern versions of the game. I am a member of the development team having contributed various features/fixes.

Maze Bounce

It's a mobile game built in Unity that involves players placing corners in randomly generated mazes of different sizes to launch a ball through the maze to reach the end.

CC-117

This is a Discord bot I made for my Wynncraft Guild. It reads data from the Wynncraft public API and uses that to perform various different features.

Accessible Codenames

This was my final year project for my Software Engineering course at Swansea University. I recreated the board game Codenames as an Android application with a strong focus on accessibility.

Myth Chase

The coursework for my University module, 'Introduction to Video Games Programming'. I took the game Pac-man, recreated it in Unity with custom levels and characters.

Collaborative Contributions

These are projects I am not actively involved in the development of, but I have contributed to it in some way.

Wynntils Legacy

Wynntils Legacy is the original version of Wynntils. For one final release I added a screen to export favourties and waypoints from this mod so that they can be imported into Wynntils.

Balls and Mines

This project was a game jam made by some friends. I did not participate in the original development but I later added simple multiplayer to it so that you could play with 4 players online.

RemoveHUD

RemoveHUD is a Minecraft mod that allows you to hide or reposition parts of your HUD. I ported the mod to 1.20.2, 1.21 and added a few new features.

ControlifyWynn

ControlifyWynn is a mod that allows users to play Wynncraft using a controller via the Controlify mod. It hooks into Artemis so I updated the mod to support the latest version of Artemis.

CustomScoreboard

CustomScoreboard is a mod that allows users to customise the Minecraft scoreboard by hiding specific lines or the title as well as adjusting the background opacity. I ported the mod to 1.21.
