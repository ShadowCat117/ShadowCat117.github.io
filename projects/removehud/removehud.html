RemoveHUD

Minecraft HUD customization mod for Fabric

Projects About CV GitHub Contact

RemoveHUD is a Fabric Minecraft mod that allows players to hide or reposition HUD elements. It offers control over hotbar placement, overlays, tooltips, experience bars, and other on-screen UI components to create a cleaner or more personalized interface.

I contributed by porting the mod to newer Minecraft versions and expanding its configuration options to support new elements introduced in later updates.
